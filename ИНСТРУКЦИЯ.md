# 🎯 Обратная связь - Квиз, плиз!

Приложение для macOS для обработки форм обратной связи с **автоматическим распознаванием текста** на основе нейросети EasyOCR.

## 📦 Установка

1. Откройте `dist/quiz-feedback-app-1.0.0-arm64.dmg`
2. Перетащите приложение в **Applications**

Или запускайте через Terminal:
```bash
npm start
```

## 🚀 Как использовать

### 1. Запустите приложение

Откройте приложение из Applications или через `npm start`

### 2. Перетащите фото формы

Перетащите изображение формы обратной связи в область Drag & Drop

### 3. Дождитесь распознавания

Приложение автоматически распознает:
- ✅ **Название команды** (из верхней части формы)
- ✅ **Комментарий** (из нижней части формы)
- ⚠️ **Оценки** - установите вручную (по умолчанию 5)

### 4. Введите название игры

Например: "Квиз, плиз! №123"

### 5. Проверьте и исправьте данные

**Важно:** Рукописный текст может распознаваться с ошибками. Всегда проверяйте:
- Название команды
- Комментарий
- Оценки (выставляются вручную)

### 6. Отправьте в Google Таблицу

Нажмите **"📊 Отправить в Google Таблицу"**

## 🛠 Технологии

- **Electron** - кроссплатформенная основа
- **EasyOCR (Python)** - нейросеть для распознавания текста
- **OpenCV** - обработка изображений
- **Google Apps Script** - интеграция с Google Таблицами

## ⚠️ Ограничения

### Рукописный текст
Нейросеть EasyOCR хорошо распознает печатный текст, но с рукописным могут быть ошибки:
- "Слишком быстро, вообще прям" → "С4( (oM Ф0 c7о"
- "Четвертичка" → "Че 7 @e/ 7w2o"

**Решение:** Всегда проверяйте распознанные данные перед отправкой!

### Оценки
Автоматическое распознавание оценок ненадежно, потому что обведенные кружочки нейросеть видит как "10".

**Решение:** Выставляйте оценки вручную через слайдеры (1-10).

## 🔧 Устранение проблем

### "Python не найден"
Приложению требуется Python 3. Установите Python с https://www.python.org/

### "EasyOCR не установлен"
Установите EasyOCR командой:
```bash
pip3 install easyocr
```

### Ошибка отправки в Google Таблицу
- Проверьте интернет-соединение
- Убедитесь, что Google Apps Script развернут правильно

### Приложение не запускается
- Убедитесь, что установлен Node.js 18+
- Выполните `npm install` в папке приложения
