# 🚀 Быстрая настройка Google Таблицы

## Шаг 1: Откройте Apps Script

1. Откройте таблицу: https://docs.google.com/spreadsheets/d/1Q7uWL-Dbr1w6szky-ATmDa6yPofzGclbP8aAIYIdYWU/edit
2. В меню: **Расширения → Apps Script**

## Шаг 2: Вставьте скрипт

Скопируйте весь код из файла `google-apps-script.js` и вставьте в редактор Apps Script

## Шаг 3: Сохраните

Нажмите 💾 **Сохранить** и назовите проект "Quiz Feedback"

## Шаг 4: Разверните веб-приложение

1. Нажмите синюю кнопку **Разместить развёртывание** → **Новое развёртывание**
2. Нажмите ⚙️ и выберите **Веб-приложение**
3. Заполните:
   - **Описание**: `Quiz Feedback API`
   - **Выполнять от**: **Меня** (ваш email)
   - **У кого есть доступ**: **Все** (или **Все, у кого есть аккаунт Google**)
4. Нажмите **Разместить**
5. Разрешите доступ при запросе

## Шаг 5: Скопируйте URL

Скопируйте **URL веб-приложения** (выглядит как `https://script.google.com/macros/s/.../exec`)

## Шаг 6: Вставьте URL в приложение

1. Откройте файл `renderer.js`
2. Найдите строку ~285:
   ```javascript
   const scriptUrl = 'YOUR_GOOGLE_APPS_SCRIPT_URL';
   ```
3. Замените на ваш URL:
   ```javascript
   const scriptUrl = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec';
   ```
4. Сохраните файл

## Шаг 7: Готово!

Запустите приложение:
```bash
npm start
```

Теперь данные будут отправляться в вашу Google Таблицу! 🎉

---

## 🔧 Если что-то пошло не так

### Ошибка "Нет доступа"
- Убедитесь, что выбрали "Все" в настройках доступа при развёртывании
- Проверьте, что таблица доступна по ссылке

### Данные не отправляются
- Откройте консоль приложения (Cmd+Option+I) и проверьте ошибки
- Убедитесь, что URL правильно скопирован
- Проверьте интернет-соединение

### Скрипт не работает
- Убедитесь, что скопировали весь код из `google-apps-script.js`
- Проверьте, что SHEET_ID правильный: `1Q7uWL-Dbr1w6szky-ATmDa6yPofzGclbP8aAIYIdYWU`
