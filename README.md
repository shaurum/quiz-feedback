# Обратная связь - Квиз, плиз!

Приложение для macOS для автоматической обработки форм обратной связи с распознаванием текста через Google Gemini AI.

## 📋 Возможности

- **Drag & Drop** - перетащите изображение формы в приложение
- **OCR распознавание** - автоматическое распознавание названия команды и комментария
- **Распознавание оценок** - определение оценок 1-10 по категориям
- **Google Gemini 2.5 Flash** - основное распознавание через мультимодальную модель Google
- **Резервные варианты**: PaddleOCR, Ollama, EasyOCR
- **Google Таблицы** - отправка данных в вашу таблицу
- **История** - сохранение истории всех отправок

## 🚀 Установка

### 1. Google Gemini API (основной метод)

API ключ уже настроен в приложении. Для использования своего ключа:

1. Получите ключ на https://aistudio.google.com/apikey
2. Установите переменную окружения:
   ```bash
   export GEMINI_API_KEY='your_api_key_here'
   ```

### 2. Опционально: Ollama (локальный резерв)

```bash
# Установка через Homebrew
brew install ollama

# Запуск сервиса
brew services start ollama

# Установка модели
ollama pull moondream
```

### 3. Установка зависимостей приложения

```bash
npm install
```

### 4. Запуск приложения

```bash
npm start
```

### 5. Сборка приложения для macOS

```bash
npm run build
```

Приложение появится в папке `dist/` как `.dmg` файл.

## ⚙️ Настройка Google Sheets API

### Вариант 1: Google Apps Script (рекомендуется)

1. Откройте вашу Google Таблицу: https://docs.google.com/spreadsheets/d/1Q7uWL-Dbr1w6szky-ATmDa6yPofzGclbP8aAIYIdYWU/edit

2. В меню выберите: **Расширения → Apps Script**

3. Удалите весь код в редакторе и вставьте содержимое файла `google-apps-script.js`

4. Нажмите **Сохранить** (💾)

5. Нажмите **Разместить развёртывание** → **Новое развёртывание**

6. Выберите тип: **Веб-приложение**

7. Настройте:
   - Описание: `Quiz Feedback API`
   - Выполнять от: **Меня**
   - У кого есть доступ: **Все** (или **Все, у кого есть аккаунт Google**)

8. Нажмите **Разместить**

9. Скопируйте **URL веб-приложения**

10. Откройте `renderer.js` и замените строку:
    ```javascript
    const scriptUrl = 'YOUR_GOOGLE_APPS_SCRIPT_URL';
    ```
    на ваш URL:
    ```javascript
    const scriptUrl = 'https://script.google.com/macros/s/.../exec';
    ```

### Вариант 2: Google Sheets API с Service Account

1. Создайте проект в [Google Cloud Console](https://console.cloud.google.com/)

2. Включите **Google Sheets API**

3. Создайте **Service Account** и скачайте JSON-ключ

4. Предоставьте доступ сервисному аккаунту к вашей таблице

5. Используйте библиотеку `googleapis` для отправки данных

## 📖 Как использовать

1. **Запустите приложение**
   ```bash
   npm start
   ```

2. **Перетащите изображение формы** в область Drag & Drop

3. **Дождитесь распознавания** - приложение автоматически определит:
   - Название команды
   - Оценки по категориям (1-10)
   - Комментарий

4. **Введите название игры** (например, "Квиз, плиз! №123")

5. **Проверьте данные** и при необходимости отредактируйте

6. **Нажмите "Отправить в Google Таблицу"**

7. **Готово!** Данные сохранены в таблице и истории

## 📊 Категории оценок

- **Сложность вопросов** (1 - очень легко, 10 - очень сложно)
- **Насколько понравились вопросы** (1 - вообще нет, 10 - прям кайф)
- **Организация игры** (1 - плохо, 10 - чётко)
- **Работа ведущего** (1 - увольняем, 10 - даём премию)
- **Обслуживание бара** (1 - отвратительно, 10 - восхитительно)
- **Общее впечатление от вечера** (1 - печально, 10 - великолепно)

## 🛠 Технологии

- **Electron** - кроссплатформенная основа
- **Google Gemini 2.5 Flash** - мультимодальная модель Google для OCR
- **PaddleOCR** - резервное распознавание текста
- **Ollama (moondream)** - локальная модель для резервного распознавания
- **EasyOCR** - последний резервный вариант
- **Google Apps Script** - интеграция с Google Таблицами
- **HTML/CSS/JS** - интерфейс приложения

## ⚠️ Примечания

- Gemini API требует интернет-соединение
- API ключ уже настроен в приложении
- Бесплатный лимит: 15 запросов в минуту
- Для лучшего распознавания делайте качественные фотографии при хорошем освещении
- Рукописный текст может распознаваться с ошибками - проверяйте данные

## 🔧 Устранение проблем

### Ошибка Gemini API
- Проверьте интернет-соединение
- Убедитесь, что API ключ действителен
- Проверьте лимиты: https://aistudio.google.com/apikey

### Ollama не подключается
- Убедитесь, что Ollama запущен: `brew services list`
- Проверьте модель: `ollama list` (должна быть moondream)

### OCR плохо распознаёт текст
- Убедитесь, что фото четкое и хорошо освещено
- Избегайте теней и бликов на форме
- Держите камеру параллельно форме

### Ошибка отправки в Google Таблицу
- Проверьте, что URL веб-приложения правильно настроен
- Убедитесь, что у вас есть доступ к таблице
- Проверьте интернет-соединение

### Приложение не запускается
- Убедитесь, что установлен Node.js 18+
- Выполните `npm install` заново
- Проверьте консоль на ошибки
